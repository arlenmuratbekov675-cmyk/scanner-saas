# Optimized Messages (Test These)

## Core Principle

**Don't sell AI. Sell the pain.**

❌ "We help clinics get more patients" → sounds like 1000 others
✅ "Do you lose patients from missed calls?" → specific, personal, engaging

---

## LinkedIn Connection Request

```
Hi {name},

quick question — do you ever lose patients from missed calls or after-hours inquiries at {clinic}?
```

**Why it works:**
- Question (engages)
- Specific pain (missed calls)
- Easy to answer
- Pattern interrupt (not "I help...")

---

## First Message (After They Accept)

```
Thanks for connecting, {name}.

Most dental clinics I looked at lose 10–30% of potential patients because calls come in after hours or don't get followed up properly.

Is that something you notice at {clinic}?
```

**Why it works:**
- Specific stat (10-30%)
- Concrete scenario (after hours, no follow-up)
- Asks about THEIR experience
- Doesn't mention AI or solution

---

## Follow-Up Messages

### Follow-Up 1 (3 days later):
```
Hey {name}, just bumping this up — is after-hours lead 
capture a challenge for {clinic}?
```

### Follow-Up 2 (5 days later):
```
Hi {name}, one more follow-up.

If this isn't a priority right now, totally understand.

If it is, happy to show you how we help clinics 
recover 15-20 extra appointments per month.
```

---

## Response Scripts

### If they say "Voicemail" / "We miss calls":
```
That's exactly what we fix.

Our system answers in 60 seconds and books the appointment 
directly into your calendar.

Clinics using this recover 15-20 extra patients per month.

Want a quick 10-min demo?
```

### If they say "We have a system":
```
Good. Quick question — does it actually book the appointment, 
or just take a message?
```

### If they say "How much?":
```
$1,500/month. But let me ask — how much is a missed patient 
worth to {clinic}?

If it's $500+, and we recover 3 per month, you're making 
money from day one.

Want to see the math for {clinic}?
```

### If they say "Send info":
```
I will. But real quick — what questions do you have right now?

I want to send you exactly what helps you decide.
```

### If they say "Not interested":
```
No problem. Can I ask — is it because you already have a 
solution, or because you don't see the need?

Just curious so I'm not wasting your time.
```

### If they don't reply to first message:
**Wait 3 days, then:**
```
Hey {name}, just following up — is capturing after-hours 
leads a challenge for {clinic}?
```

## Rules

1. **No "AI" in first message** — they don't care about tech
2. **Under 50 words** — short messages get replies
3. **One question only** — don't overwhelm
4. **Personalize** — use their name and clinic name
5. **Track everything** — you need data to optimize

---

## What Makes These Different

❌ Old: "I help dental clinics get more booked patients"
→ Sounds like everyone else

✅ New: "Do you ever lose patients from missed calls or after-hours inquiries?"
→ Specific pain, personal, easy to answer

❌ Old: "We use AI to automate..."
→ They don't care

✅ New: "Most clinics lose 10-30% of potential patients because calls come in after hours"
